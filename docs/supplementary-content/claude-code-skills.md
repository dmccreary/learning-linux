# Claude Code Skills

Claude Skills were announced by Anthropic on October 16, 2025.

According to https://simonwillison.net/2025/Oct/16/claude-skills/, Skills are "conceptually extremely simple: a
skill is a Markdown file telling the model how to do something, optionally accompanied by extra documents and
pre-written scripts that the model can run to help accomplish the tasks."

Key details from the https://www.anthropic.com/news/skills:
- Skills are modular, reusable task components
- They work across Claude apps, Claude Code, and the API
- Available on Pro, Max, Team, and Enterprise plans
- Skills can include executable code for reliable task completion
- Developers can install skills from the anthropics/skills marketplace
